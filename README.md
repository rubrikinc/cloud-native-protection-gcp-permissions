# Permissions Sets - Cloud-Native Protection for GCP

# Azure Role Definition: Native Protection

This repository contains the different permission sets that are associated with the Polaris role when protecting customer workloads in GCP projects using Rubrik Security Cloud. These role definitions are not intended for use outside of Rubrik Polaris.

# :white_check_mark: Prerequisites

There are a few services you'll need in order to get this project off the ground:

* GCP Service Enablements - enable services specified with the permission set to be able to exercise the privileges
* Rubrik Polaris - the platform that protects provisioned workloads
